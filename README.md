## 🎯 Objectif

Créer une version moderne et optimisée du site "Installations Électriques" pour les enseignants du domaine 3 - Électricité.

## ✨ Fonctionnalités Modernes

- ✅ Design responsive (mobile, tablette, desktop)
- ✅ Bulle d'hommage "Merci Denis" avec lien vers site original
- ✅ Formules mathématiques avec MathJax
- ✅ Dessins SVG interactifs et animés
- ✅ Navigation moderne et intuitive
- ✅ Optimisé pour GitHub Pages

## 🚀 Utilisation

### Développement Local

```powershell
# Lancer un serveur HTTP local
python -m http.server 8000

# Ouvrir dans le navigateur
http://localhost:8000/index.html
```

### Déploiement GitHub Pages

1. Copier le contenu de ce dossier dans un repository GitHub
2. Activer GitHub Pages dans les paramètres du repository
3. Le site sera accessible à l'URL : `https://[username].github.io/[repo-name]/`
